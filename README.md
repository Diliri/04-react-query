# 🎬 React Movies App (with TanStack Query & Pagination)

Цей вебдодаток — інтерактивний пошуковик фільмів із пагінацією, створений на базі бібліотеки **React** та **TypeScript** у рамках навчання в GoIT. Додаток інтегрується з API сервісу **The Movie Database (TMDB)** та використовує сучасні інструменти для ефективного менеджменту стану запитів.

🚀 **Живий лінк на проєкт:** [04-react-query](https://03-react-movies-sable-kappa.vercel.app/)

---

## 🔍 Основний функціонал (Features)

- **Динамічний пошук:** швидкий пошук кінострічок за ключовими словами чи назвою через інтерактивний рядок пошуку.
- **Зручна пагінація:** інтеграція бібліотеки `react-paginate` дозволяє комфортно переглядати великі списки результатів по сторінках.
- **Галерея результатів:** відображення знайдених фільмів у вигляді адаптивної сітки (картки містять постери, назви та глядацькі рейтинги).
- **Детальна інформація:** клік по картці відкриває модальне вікно з детальним описом (`overview`), датою релізу, середньою оцінкою та великим банером.
- **Валідація та помилки:** додаток коректно обробляє ситуації порожнього запиту (через спливаючі сповіщення `react-hot-toast`), відсутності результатів або збою мережі.
- **Зручність інтерфейсу:** під час завантаження відображається лоадер, а при відкритті модалки блокується фоновий скрол сторінки.

## 🛠 Технологічний стек (Tech Stack)

- **Frontend:** React (Функціональні компоненти + Hooks)
- **Менеджмент серверного стану:** TanStack Query (`@tanstack/react-query`) — керує кешуванням, автоматичними запитами при зміні сторінок та станами завантаження/помилки.
- **Мова програмування:** TypeScript (сувора типізація інтерфейсів API, пропсів компонентів та подій)
- **Збирач проєкту:** Vite
- **Стилізація:** CSS Modules, `modern-normalize`
- **HTTP-клієнт:** Axios (із налаштуванням глобального `instance` та Bearer авторизації)
- **Деплой:** Vercel

## 📂 Структура проєкту (Key Architecture)

Проєкт побудований за модульним принципом рознесення компонентів у власні папки поруч зі стилями:

- `src/components/` — ізольовані React-компоненти (`SearchBar`, `MovieGrid`, `MovieModal`, `Loader`, `ErrorMessage`, `App`), кожен у своїй однойменній папці з файлом стилів `.module.css`.
- `src/services/` — сервіс `movieService.ts` для ізоляції логіки запитів до API TMDB.
- `src/types/` — глобальні файли типізації даних (наприклад, `src/types/movie.ts` для моделей фільму та відповідей сервера).
